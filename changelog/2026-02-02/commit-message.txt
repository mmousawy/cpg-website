feat: Photo Challenges feature

Complete implementation of photo challenges - themed prompts where admins create challenges,
members submit photos (upload or from library), and submissions require admin approval.

Database:
- Add challenges, challenge_submissions, challenge_announcements tables
- Add challenge_photos view for accepted submissions
- Add RPC functions: submit_to_challenge, review_challenge_submission, bulk_review_challenge_submissions
- Add RLS policies for viewing and submitting
- Add max_photos_per_user limit option per challenge
- Add challenge stats to get_user_stats and get_profile_stats RPCs
- Add admin_notifications email type for admin-only preferences
- Prevent resubmission of rejected photos

Public Pages:
- /challenges - List of active and past challenges
- /challenges/[slug] - Challenge detail with gallery, comments, and submit button

User Features:
- Submit photos to challenges (drag-and-drop upload or from library)
- Uploads default to public for challenge submissions
- View submission status in /account/challenges
- Receive notifications when submissions are accepted/rejected

Admin Features:
- /admin/challenges - List all challenges with stats
- /admin/challenges/[slug] - Create/edit challenges with cover image upload
- /admin/challenges/[slug]/submissions - Review queue with accept/reject
- Bulk accept/reject with optional rejection reason
- Announce challenges via email with recipient selection
- Receive notifications on new submissions

Notifications:
- challenge_announced, new_submission, submission_accepted, submission_rejected types
- Email templates for announcements and submission results
- Admin notification email type (only visible to admins in preferences)

Components:
- ChallengeCard, ChallengeHeader, ChallengeCoverImage, ChallengeGallery
- SubmitToChallengeContent with drag-drop upload and library selection
- SubmissionSuccessContent with photo stack animation
- AnnounceChallengeModal with recipient list
- GridCheckbox reusable component for photo selection
- Profile badges for challenges participated and photos accepted
